## üß¨ The Digital Organism Architecture

AetherMind is structured into five core biological layers:

| Layer | Purpose | Biological Analogue |
|-------|---------|---------------------|
| **Brain** | Fixed reasoning engine (HOW to think) | Cerebral Cortex |
| **Mind** | Infinite-memory vector store (WHAT to think about) | Long-term Memory |
| **Heart** | Emotion, Ethics, and Moral Valencing | Limbic System |
| **Body** | Multi-interface adapters (Pluralistic Embodiment) | Motor Functions / Avatar |
| **Orchestrator** | Nervous system and cognitive cycle | Central Nervous System |

## üîÑ The Cognitive Cycle: Active Inference

Everything AetherMind does flows through a continuous **Active Inference Loop** to minimize prediction error (surprise):

1. **Sense:** Retrieve domain-weighted context from the Mind.
2. **Feel:** Compute emotional valence and moral constraints in the Heart.
3. **Reason:** The Brain processes intent using specialized domain profiles.
4. **Embellish:** Affective cues are added to the internal response.
5. **Parse:** Structured **Action Tags** are extracted for execution.
6. **Execute:** The Body performs tasks (Code, Research, Hardware).
7. **Learn:** Results are ingested into episodic memory for future cycles.
